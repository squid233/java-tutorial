# 字符类型

/// admonition | 本节前置知识
    type: tip
Unicode 编码
///

字符类型是另一种整数类型，其名称为`#!java char`。

字符类型的字面量比较简单，用两个**半角**单引号（`'`）括住一个（不多不少）字符即可，例如：`'A'`。

字符类型受限较多，只能表示`0x0000`~`0xFFFF`范围内的 UTF-16 字符（也就是双字节字符）。

## 转义

一些特殊字符不能直接用字面量表示，我们可以通过反斜杠（`\`）转义[^1]得到这些字符。

常用的转义序列如下：

| 转义序列     | 意义                        |
|----------|---------------------------|
| `\n`     | 换行                        |
| `\t`     | 制表符                       |
| `\"`     | 双引号（`"`）                  |
| `\'`     | 单引号（`'`）                  |
| `\\`     | 反斜杠（`\`）                  |
| `\uxxxx` | 一个 UTF-16 代码单元（code unit） |

[^1]: [3.10.7. Escape Sequences](https://docs.oracle.com/javase/specs/jls/se24/html/jls-3.html#jls-3.10.7)
