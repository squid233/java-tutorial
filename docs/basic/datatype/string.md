# 字符串

字符串不属于基本类型，而是我们之后会认识的引用类型。

字符串（`String`）简单来说就是一串文本，其字面量由两个成对的**半角**双引号（`"`）以及被括起来的文本组成，如下。

```java
"Hello world!"  // 这是一个字符串
"123"           // 这是一个字符串。这不是数字
"   "           // 这是一个只有空格的字符串
""              // 这是一个空字符串
"Aa\u4E00\n"    // 这是一个包含转义序列的字符串
```

关于转义序列，请见[字符类型](char.md#转义)。

## 文本块

我们可以用文本块表示多行文本，不仅直观，而且能省略大量`\n`转义。

用三个连续的成对的**半角**双引号（`"""`）来创建一个文本块。

```java
"""
Hello,
world!
This is \
not a new line
"""
```

开头的三个双引号后该行内不能有任何内容。反斜杠（`\`）后接换行可使该行与下一行连接。
