# 小结

- JDK 有不同的供应商，不同供应商开发的 OpenJDK 性能和功能各异。
- JShell 可以在不编写源文件的情况下运行一段代码。
- Java 中有 8 种基本类型，不同类型的取值范围不同。
- 基本类型和`String`有字面量来表示它们的值。
- Java 支持四则运算和取余。
- 各种逻辑运算符产生`#!java boolean`值，帮助实现流程控制。
- 标识符区分大小写。
- `#!java final`修饰的变量必须赋值一次才能使用，且不可重新赋值。
- Java 是静态类型、强类型语言，变量的类型不会改变。
- 方法用于减少重复工作。
- 变量和方法的返回值支持各种运算。
- IDE 提高了开发效率。
- 流程控制包括条件、选择、循环语句，加强程序的功能。
